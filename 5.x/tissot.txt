Step Installation :
Follow these instructions carefully:
1. Backup your internal storage files
2. Switch to Zignas' TWRP: https://github.com/zignas/twrp_device_xiaomi_tissot_treble/releases
3. Install the vendor--adder.zip file: https://drive.google.com/file/d/1Gj-eDIttfJU5Ad-ApGdL94yYp0KD3FlL/view?usp=share_link
4. Install the ROM. (Magisk is optional)

The vendor--adder.zip will only be flashed once and you can disregard it for future dirty flashes.
